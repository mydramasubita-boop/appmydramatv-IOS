X MODIFICHE
# 1. Installa dipendenze
npm install

# 2. Build web
npm run build

# 3. Init Capacitor
npx cap init MyDramaLife com.mydramalife.mobile --web-dir=dist

# 4. Aggiungi Android
npx cap add android


# 5. Sostituisci file
android/app/src/main/res/values/styles.xml → nel file zip
android/app/src/main/AndroidManifest.xml → nel file zip
android/app/src/main/java/com/mydramalife/mobile/MainActivity.kt → nel file zip

# 6. Patch automatiche (MainActivity.kt, manifest, icona)
npm run cap:setup

# 7. Copia web in android
npx cap copy android

# 8. Apri Android Studio
npx cap open android
